### 1.1 shutdown

- `shutdown`命令可以 安全 关闭 或者 重新启动

| 选项 | 含义     |
| ---- | -------- |
| `-r` | 重新启动 |
| `-c` | 取消关机/重启         |

>提示：
>- **不指定选项和参数**，默认表示**1分钟**之后关闭电脑
>- 远程维护服务器时，最好不要关闭系统，而应该重新启动系统

- 常用命令示例

```linux
# 重新启动操作系统，其中 now 表示现在
$ shutdown -r now

# 立即关机，其中 now 表示现在
$ shutdown now

# 系统在今天的 20：25 关机
$ shutdown 20:25

# 系统再过十分钟后自动关机
$ shutdown +10

# 取消之前指定的关机计划
$ shutdown -c
```
