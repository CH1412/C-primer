## Linux终端命令格式

### 01.终端命令格式
`command [-options] [parameter]`
- `command` :命令名，相应功能的英文单词或单词的缩写
- `[-options]` :选项，可用来对命令进行控制，也可以省略
- `parameter` :传给命令的参数，可以是0个、1个或多个

>`[]`表示可选

### 0.2查询命令帮助信息
1. --help
	`command --help`
	显示`command`命令的帮助信息
2. man
	`man command`
	查阅`command`命令的使用手册
	>`man`是manual的缩写，是Linux提供的一个手册，包含了绝大部分的命令、函数的详细使用说明

使用`man`时的操作键：

| 操作键  | 功能                 |
| ------- | -------------------- |
| 空格键  | 显示手册页的下一屏   |
| Enter键 | 一次滚动手册页的一行 |
| b       | 回滚一屏             |
| f       | 前滚一屏             |
| q       | 推出                 |
| /word   | 搜索 word 字符串                     |
